R Packages Link to heading

At IPEA, while working on Access to Opportunities project, I collaborated extensively on developing R packages that utilize GTFS data. For some packages, I contributed directly to the coding phase, and for all, I actively participated in discussions, suggested improvements, and reported bugs.

In addition, I have started developing my own R packages. One package currently in development is gps2gtfs, which uses public transport GPS data to correct GTFS feeds based on scheduled data. This package applies the method I developed in this paper as part of my PhD research.
